Trabajos

  • [01] Diseño de marca

    Proyectos de identidad de marca con alto rendimiento y calidad gráfica. A través de un enfoque estratégico y creativo, desarrollamos identidades visuales que destacan en el mercado y comunican eficazmente sus valores y misión. Desde creación de marcas gráficas, hasta rediseños, garantizamos soluciones integrales que impulsan el reconocimiento y el reposicionamiento de la marca.

  • [02] Diseño UX/UI

    En el diseño UX/UI, me enfoco en crear experiencias intuitivas y atractivas para los usuarios. Cada proyecto comienza con una profunda investigación del usuario para entender sus necesidades y comportamientos. A partir de ahí, desarrollo wireframes y prototipos que facilitan la interacción y mejoran la usabilidad.

  • [03] Desarrollador Front End

    interfaces web atractivas y funcionales que brindan una experiencia de usuario excepcional. Utilizo tecnologías como HTML, CSS y JavaScript para desarrollar soluciones eficientes y responsivas, asegurando que cada sitio web no solo se vea bien, sino que también funcione de manera óptima en diferentes dispositivos y navegadores.
